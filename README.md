# python-cybersecurity-tools

A Python script that scans web server logs for suspicious request patterns using regular expressions.

## Features
- Detects wildcard characters in URIs
- Detects backtick usage
- Detects possible code injection patterns
- Detects possible CSS/script injection
- Detects folder traversal attempts
- Reads local log files and prints flagged entries

## Skills Used
- Python
- Regular expressions
- File I/O
- Cybersecurity log analysis
- Basic threat detection

## How to Run
python security_log_analyzer.py

Then enter the log file name when prompted.
